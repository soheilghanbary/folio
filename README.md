## Soheil Ghanbary Portfolio

Hi, This is a Sample Portfolio built with Next.js and Tailwind CSS.

```bash
bun install
```

Run the development server:

```bash
bun dev
```

Open [http://localhost:3000](localhost:3000) with your browser to see the result 🚀.
